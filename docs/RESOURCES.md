# Learning Resources

## MITRE ATT&CK

- [MITRE ATT&CK Framework](https://attack.mitre.org/) — Adversary tactics and techniques
- [ATT&CK Navigator](https://mitre-attack.github.io/attack-navigator/) — Visual coverage heatmap
- [D3FEND](https://d3fend.mitre.org/) — Defensive techniques mapped to ATT&CK

## Cryptography

- [Cryptography.io docs](https://cryptography.io/en/latest/) — Python `cryptography` library
- [Serious Cryptography](https://nostarch.com/seriouscrypto) — Jean-Philippe Aumasson (book)
- [NIST Post-Quantum Standards](https://www.nist.gov/pqcrypto) — Future-proof algorithms

## Web Security

- [OWASP Top 10](https://owasp.org/Top10/) — Most critical web application risks

- [PortSwigger Web Security Academy](https://portswigger.net/web-security) — Free interactive labs
